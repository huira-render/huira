Naming Conventions
------------------

**C++ Naming:**

* Classes: ``PascalCase``
* Functions/methods: ``snake_case``
* Private members: ``snake_case_``
* Constants: ``SCREAMING_SNAKE_CASE``
* Namespaces: ``lowercase``


**Python Naming:**

* Classes: ``PascalCase``
* Functions/methods: ``snake_case``
